该项目是Keras博客https://blog.keras.io/building-powerful-image-classification-models-using-very-little-data.html
的练习，参考了博客中的源码，项目中需要用到的数据可以在下面网址中下载 https://www.kaggle.com/c/dogs-vs-cats/data 。
实验平台：Ubuntu16.04+Python2.7+Keras2.0.8(using Tensorflow-cpu backend)
原始博客作者Francois Chollet（Keras的发明者）在博客中提供的源码有些许过时的地方，直接参照运行可能会报错，具体错误可谷歌或百度解决，也可参照本项目的代码（已调通）。
